Maze

GitHub Link

This is a mazegenerator I created to improve my java skills and understanding in graph theory. It randomly generates mazes with dimensions 5x5 - 10x10 using a randomized depth search algorithm. The GUI is handled by Java swing and the project uses Maven package managing.
